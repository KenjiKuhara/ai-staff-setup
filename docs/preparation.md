# 事前準備ガイド

このガイドでは、AI社員セットアップの前に必要な準備を説明します。
パソコンに詳しくない方でも、この手順通りに進めれば大丈夫です。

---

## 準備するもの

- Windows 11 のパソコン
- インターネット接続
- メールアドレス（Anthropicアカウント作成に使います）

所要時間: 約10分

---

## Step 1: Anthropic アカウントを作成する

AI社員を動かすために、Anthropic（アンソロピック）という会社のアカウントが必要です。
これはAIの「雇用契約」のようなものだと思ってください。

### 1-1. アカウント作成ページを開く

以下のURLをブラウザで開いてください。

```
https://console.anthropic.com/
```

### 1-2. アカウントを作成する

1. 「Sign up」をクリック
2. メールアドレスを入力して「Continue」をクリック
3. メールに届いた確認コードを入力
4. 名前を入力して登録完了

### 1-3. 支払い方法を登録する（クレジットカード）

AI社員を動かすには利用料がかかります（従量課金制。使った分だけ請求されます）。
目安: 1日30分ほどの利用で月2,000〜5,000円程度です。

1. 左メニューの「Settings」→「Billing」を開く
2. 「Add payment method」をクリック
3. クレジットカード情報を入力
4. 利用上限（Spend limit）を設定する（最初は$20程度がおすすめ）

---

## Step 2: セットアップスクリプトを実行する

### 2-1. PowerShell を開く

1. キーボードの **Windows キー** を押す（画面左下のWindowsマーク）
2. `powershell` と入力する
3. 「Windows PowerShell」が表示されるのでクリックして開く

黒い（または青い）画面が開きます。これが「ターミナル」です。

### 2-2. 作業フォルダーを作る

ターミナルに以下を1行ずつコピーして貼り付け、Enterを押してください。

```powershell
mkdir $HOME\ai-staff
```

```powershell
cd $HOME\ai-staff
```

これで「ai-staff」という名前のフォルダーが作られ、その中に移動しました。

### 2-3. セットアップを実行する

以下の1行をコピーして貼り付け、Enterを押してください。

```powershell
iex(irm 'https://raw.githubusercontent.com/KenjiKuhara/ai-staff-setup/main/setup.ps1')
```

自動的にインストールが始まります。途中で「管理者権限の確認」画面が出たら「はい」を押してください。

以下が自動でインストールされます:
- VS Code（プログラムの編集ツール。既にインストール済みならスキップされます）
- Node.js（Claude Codeの動作に必要。既にインストール済みならスキップされます）
- Claude Code（AI社員の本体）
- AI社員の初期ファイル（CLAUDE.md、秘書室フォルダー）

### 2-4. 「PATH に反映されませんでした」と出た場合

ターミナルを閉じて、Step 2-1 からやり直してください。
2回目の実行では、既にインストールされたものはスキップされるので安心してください。

---

## Step 3: AI社員を起動する

### 3-1. VS Code でフォルダーを開く

1. VS Code を起動する（デスクトップやスタートメニューから）
2. メニュー「ファイル」→「フォルダーを開く」
3. 先ほど作った `ai-staff` フォルダーを選択（場所: `C:\Users\あなたのユーザー名\ai-staff`）

### 3-2. ターミナルを開く

1. VS Code のメニュー「ターミナル」→「新しいターミナル」
2. 画面の下半分にターミナルが開きます

### 3-3. Claude Code を起動する

ターミナルに以下を入力してEnter:

```
claude
```

初回はログイン画面が表示されます。
Step 1 で作成した Anthropic アカウントでログインしてください。

### 3-4. 秘書を起動する

ログインが完了したら、以下を入力してEnter:

```
company
```

秘書が挨拶をしたら、セットアップ完了です。おめでとうございます。

## よくある質問

### Q: 料金はいくらかかりますか?

従量課金制です。使った分だけ請求されます。
目安として、1日30分ほどの利用で月2,000〜5,000円程度です。
Step 1-3 で設定した利用上限を超えることはありません。

### Q: セットアップに失敗しました

ほとんどの場合、ターミナルを閉じて開き直し、もう一度セットアップコマンドを実行すれば解決します。
既にインストールされたものは自動的にスキップされます。

### Q: 秘書に何を聞けばいいですか?

「最近、面倒だなと感じている仕事はありますか?」と秘書から聞いてくれます。
思いつくままに答えてください。秘書があなたの仕事を整理してくれます。

### Q: やめたくなったらどうすればいいですか?

VS Code のターミナルで `/exit` と入力するか、ターミナルを閉じれば終了します。
Anthropic の利用上限を $0 に設定すれば課金も止まります。
インストールしたものを削除したい場合は、以下のコマンドで削除できます:

```powershell
npm uninstall -g @anthropic-ai/claude-code
```
