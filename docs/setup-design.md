# セットアップスクリプト（setup.ps1）詳細設計書

## このスクリプトは何をするのか

このスクリプトは、あなたのパソコンに「AI社員」を迎え入れるための準備を自動で行います。具体的には、AI社員が働くために必要なソフトウェア3つ（VS Code、Node.js、Claude Code）がパソコンに入っているか確認し、なければインストールします。最後に、AI社員が使うフォルダとファイルを作成します。すべての処理はあなたのパソコンの中だけで完結し、外部に情報を送ることはありません。

---

## 実行前の状態と実行後の状態

| 項目 | 実行前 | 実行後 |
|------|--------|--------|
| VS Code（文章やプログラムを書くためのアプリ） | 入っていない場合がある | インストール済み |
| Node.js（Claude Codeを動かすための土台） | 入っていない場合がある | インストール済み |
| Claude Code（AI社員の本体） | 入っていない場合がある | インストール済み |
| CLAUDE.md（AI社員の設定書） | 存在しない | 作成済み |
| .companyフォルダ（AI社員の作業場所） | 存在しない | 作成済み |

既にインストール済みのソフトウェアはスキップされます。パソコンの既存の設定や他のアプリに影響はありません。

---

## 処理の詳細

### 前提チェック: winget の確認

スクリプトが最初に行うのは、winget（ウィンゲット）というWindowsの標準ツールが使えるかの確認です。wingetは「ソフトウェアを安全にインストールするための窓口」のような役割です。

- **wingetがある場合**: そのまま次に進みます
- **wingetがない場合**: 「Microsoft Storeで『アプリ インストーラー』を更新してください」というメッセージが表示され、スクリプトは停止します。パソコンには何も変更されません

> Windows 10（バージョン1809以降）とWindows 11には標準で入っています。表示された場合は、Microsoft Storeアプリを開いて更新するだけで解決します。

---

### Step 1: VS Code（ブイエスコード）

VS Codeは、AI社員と会話するための画面を提供するアプリです。文章を書いたりファイルを見たりするのに使います。

- **何をチェックしているか**: パソコンに `code` というコマンド（VS Codeの呼び出し名）が使えるかを確認しています
- **既にインストール済みの場合**: 「VS Code （バージョン番号） ... OK」と表示され、何もせず次に進みます
- **新規インストールの場合**:
  1. wingetを使ってMicrosoft公式のVS Codeをダウンロード、インストールします
  2. 管理者権限の確認画面（「このアプリがデバイスに変更を加えることを許可しますか?」）が表示されたら「はい」を押してください
  3. インストール後、バージョン番号が表示されれば成功です
- **失敗した場合**: エラーメッセージと手動インストール用のURL（https://code.visualstudio.com/）が表示され、スクリプトは停止します。この時点でパソコンに悪影響はありません
- **パソコンへの影響**: VS Codeが `C:\Users\（ユーザー名）\AppData\Local\Programs\Microsoft VS Code\` にインストールされます。スタートメニューにも追加されます

---

### Step 2: Node.js（ノードジェイエス）

Node.jsは、Claude Code（AI社員の本体）を動かすために必要な土台のソフトウェアです。直接操作することはありませんが、裏側で必要です。

- **何をチェックしているか**: パソコンに `node` というコマンドが使えるかを確認しています
- **既にインストール済みの場合**: 「Node.js （バージョン番号） ... OK」と表示され、何もせず次に進みます
- **新規インストールの場合**:
  1. wingetを使って公式のNode.js LTS版（長期サポート版、安定版のこと）をダウンロード、インストールします
  2. 管理者権限の確認画面が表示されたら「はい」を押してください
  3. インストール後、バージョン番号が表示されれば成功です
- **失敗した場合**: エラーメッセージと手動インストール用のURL（https://nodejs.org/）が表示され、スクリプトは停止します。Step 1で入れたVS Codeはそのまま残りますが、問題ありません
- **パソコンへの影響**: Node.jsが `C:\Program Files\nodejs\` にインストールされます。npm（エヌピーエム）というソフトウェア管理ツールも一緒に入ります

---

### Step 3: Claude Code（クロードコード）

Claude Codeは、AI社員の本体です。Anthropic社が提供するAIツールで、あなたとAI社員が会話するための仕組みです。

- **何をチェックしているか**: パソコンに `claude` というコマンドが使えるかを確認しています
- **既にインストール済みの場合**: 最新版への更新を試み、「Claude Code ... OK (最新版)」と表示されます
- **新規インストールの場合**:
  1. Step 2で入れたnpm（ソフトウェア管理ツール）を使って、Anthropic社の公式パッケージをダウンロード、インストールします
  2. インストールの進行状況が画面に表示されます
  3. 「Claude Codeをインストールしました。」と表示されれば成功です
- **失敗した場合**: エラーメッセージが表示され、スクリプトは停止します。「ターミナルを閉じて開き直してから、もう一度実行してください」と案内されます
- **パソコンへの影響**: Claude Codeが npmのグローバルフォルダ（`C:\Users\（ユーザー名）\AppData\Roaming\npm\`）にインストールされます

---

### Step 4: AI社員の初期ファイルを作成

AI社員が使うフォルダとファイルを、スクリプトを実行したフォルダの中に作成します。

- **何をチェックしているか**: 各フォルダやファイルが既に存在するかを確認しています
- **既に存在する場合**: 「スキップ」と表示され、上書きしません。あなたが既に編集した内容はそのまま残ります
- **新規作成の場合**: フォルダとファイルを作成し、「作成」と表示されます
- **失敗した場合**: フォルダの書き込み権限がない場合にエラーになることがあります。別のフォルダで試してください
- **パソコンへの影響**: スクリプトを実行したフォルダの中にだけファイルが作られます。それ以外の場所には影響しません

---

## 作成されるファイルの説明

### CLAUDE.md

AI社員の「取扱説明書」兼「性格設定書」です。このファイルに書かれた内容をAI社員が読み、それに従って動きます。

初期状態では以下の内容が書かれています:

- **起動のしかた**: `company` または `おはよう` と入力するとAI社員が起動すること
- **起動時にやること**: 環境のチェック、受信箱の確認、前回の続きの把握
- **環境サポート**: Gitが入っていなければ案内する、バックアップを手助けする、などの技術サポート
- **AI社員の役割**: 「秘書」としてあなたの仕事をヒアリングし、AIに任せられる仕事を見つける
- **禁止事項**: ファイルを勝手に削除しない、確認なしに外部へ情報を送らない、機能を押し売りしない

このファイルは自由に書き換えることができます。AI社員の動き方を変えたいときは、このファイルを編集してください。

### .companyフォルダ

AI社員の作業場所です。以下の3つのサブフォルダが作られます。

| フォルダ | 役割 |
|----------|------|
| `.company/secretary/inbox/` | 「受信箱」です。メモ、アイデア、依頼など何でも入れておく場所です。AI社員が起動時にここを確認します |
| `.company/reviews/` | AI社員が仕事をした際の品質チェックの記録を保存する場所です |
| `.company/.trash/` | 不要になったファイルの「ごみ箱」です。AI社員はファイルを削除する代わりにここに移動します |

このほか、AI社員が起動後のヒアリングを通じて `.company/secretary/work-inventory.md`（業務棚卸しリスト）を作成していきます。

## よくある不安への回答

### 「パソコンが壊れませんか?」

壊れません。このスクリプトが行うのは、3つのソフトウェアのインストールと、フォルダ/ファイルの作成だけです。Windowsのシステム設定を変更したり、既存のアプリを書き換えたりすることはありません。万が一インストールに失敗しても、途中でスクリプトが安全に停止します。

### 「個人情報は送信されますか?」

このスクリプト自体は、外部に情報を送信しません。行うのはソフトウェアのダウンロードとインストールだけです。ダウンロード元はMicrosoft公式（VS Code、Node.js）とAnthropic公式（Claude Code）のみです。

ただし、セットアップ後にClaude Code（AI社員）を使い始めると、あなたがAI社員に伝えた内容はAnthropic社のサーバーに送信されます（AIが考えるために必要な通信です）。Anthropicのプライバシーポリシーに従って処理されます。

### 「料金は発生しますか?」

このセットアップスクリプト自体は無料です。インストールされる3つのソフトウェア（VS Code、Node.js、Claude Code）もすべて無料です。

ただし、Claude Code（AI社員）を使うにはAnthropicのアカウントが必要で、利用量に応じた料金が発生します。料金体系はAnthropicの公式サイトで確認できます。セットアップしただけでは課金は始まりません。

### 「アンインストールできますか?」

はい、すべて元に戻せます。詳しい手順はこのページの最後に記載しています。

### 「途中で止めても大丈夫ですか?」

大丈夫です。スクリプトはStep 1からStep 4まで順番に進みますが、途中でウィンドウを閉じたり、Ctrl+Cで中断しても問題ありません。途中まで入ったソフトウェアがパソコンに残りますが、悪影響はありません。もう一度スクリプトを実行すれば、既に完了したStepはスキップして続きから再開します。

## アンインストール手順

すべてを元に戻したい場合は、以下の手順でインストールしたものを順番に削除できます。必要なものだけ選んで削除しても問題ありません。

### 1. Claude Code を削除する

PowerShellで以下を実行します:

```powershell
npm uninstall -g @anthropic-ai/claude-code
```

### 2. Node.js を削除する

Windowsの「設定」から削除します:

1. スタートメニューから「設定」を開く
2. 「アプリ」を選ぶ
3. 「インストールされているアプリ」（Windows 11）または「アプリと機能」（Windows 10）を選ぶ
4. 一覧から「Node.js」を探して「アンインストール」を選ぶ

### 3. VS Code を削除する

同じく「設定」から削除します:

1. 上記と同じ手順で「アプリ」の一覧を開く
2. 「Microsoft Visual Studio Code」を探して「アンインストール」を選ぶ

### 4. 作成されたファイルを削除する

スクリプトを実行したフォルダの中にある以下を削除します:

- `CLAUDE.md` ファイル
- `.company` フォルダ（中身ごと）

通常のファイル削除（右クリックで「削除」）で問題ありません。

### 補足

- 上記の手順はこのスクリプトでインストールしたものだけを削除します。他のアプリや設定には影響しません
- VS CodeやNode.jsをこのスクリプト以前から使っていた場合は、削除せずそのまま残してください
- 削除の順番は「Claude Code → Node.js → VS Code」が推奨です（Claude CodeがNode.jsに依存しているため）
